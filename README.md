# Chrome_Dinausaur_game
a simple fun game on Arduino UnO

Components needed
1.Arduino UNO 
2.16x2 lcd screen with I2C driver
3.A push button
4.A resistor (of resistance between 1 and 10k ohms)
5.A piezo buzzer
6.Jumper Wires
7.Breadboard
8.A Battery
9.An enclosure of any sort
